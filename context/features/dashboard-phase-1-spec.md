# Dashboard UI Phase 1 Spec

## Overview

This is phase 1 of 3 for the dashboard UI. Use the dashboard screenshots referenced below for layout and visual direction.

The goal of this phase is to establish the dashboard foundation, global layout shell, and the first pass of the top-level interface structure before adding the fully styled sidebar, dashboard content, and drawer details in later phases.

## Requirements for phase 1

- ShadCN UI initialization and required base components
- Dashboard route at `/dashboard`
- Main dashboard layout shell and any needed global styles
- Light dashboard styling to match the provided screenshots
- Top navigation/header area
- Header should include:
  - current date display
  - notification icon
  - overflow/menu icon
- Placeholder for left sidebar and dashboard content area for initial scaffolding
- Add simple placeholder labels for now:
  - `Sidebar`
  - `Main Dashboard`
- Ensure layout is responsive from the beginning
