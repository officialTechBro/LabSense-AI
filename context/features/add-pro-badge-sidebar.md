# Overview

Add a premium badge to the appropriate LabSense sidebar features to indicate functionality reserved for paid users.

This badge should appear beside premium-only navigation items or feature entries in the sidebar, while preserving the current visual style of the dashboard.

## Requirements
- Use the ShadCN UI Badge component
- Keep the badge clean, subtle, and consistent with the LabSense medical-style interface
- Display PRO in all uppercase
- Only show the badge on sidebar the user name in the sidebar footer
- Do not change the existing sidebar layout or spacing more than necessary
